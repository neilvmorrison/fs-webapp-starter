# Full Stack Project Starter

A pnpm monorepo that contains a KoaJS server and a vite React web app.

## Prerequisites

- [Node.js](https://nodejs.org/en/download/) (LTS)
- [pnpm](https://pnpm.io/installation) (LTS)

## Installation

```bash
pnpm install
```

## Running the app

```bash
pnpm run dev
```
